# 📘 Miniguia de Estudos — Python para Iniciantes

## Sobre o Projeto
Este projeto foi desenvolvido como parte de um desafio da DIO com o objetivo de utilizar Inteligência Artificial como ferramenta de aprendizagem ativa.

O foco deste caderno temático é estudar os fundamentos da linguagem Python para iniciantes utilizando o NotebookLM para organizar conteúdos, gerar resumos e criar materiais de revisão.

---

# Objetivos de Estudo

- Aprender os fundamentos básicos de Python;
- Entender lógica de programação;
- Estudar variáveis, estruturas condicionais e loops;
- Compreender funções e listas;
- Utilizar IA para auxiliar nos estudos e revisões.

---

# Ferramentas Utilizadas

- Python
- NotebookLM
- GitHub
- Inteligência Artificial

# 📂 Estrutura do Projeto

O projeto contém:
