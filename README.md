## Resumos de Estudo

### Variáveis

Variáveis são espaços utilizados para armazenar informações em um programa.

Exemplo:

```python
nome = "Filipe"
idade = 25
```

---

### Estruturas Condicionais

As estruturas condicionais permitem tomar decisões dentro do código.

Exemplo:

```python
idade = 18

if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")
```

---

### Loops

Loops permitem repetir ações automaticamente.

#### Exemplo com for

```python
for numero in range(5):
    print(numero)
```

#### Exemplo com while

```python
contador = 0

while contador < 5:
    print(contador)
    contador += 1
```

---

### Funções

Funções ajudam a organizar e reutilizar código.

```python
def saudacao(nome):
    print(f"Olá, {nome}")

saudacao("Filipe")
```

---

## Glossário

| Termo       | Definição                    |
| ----------- | ---------------------------- |
| Variável    | Espaço que armazena valores  |
| Loop        | Estrutura de repetição       |
| Função      | Bloco reutilizável de código |
| Condicional | Estrutura de decisão         |
| Python      | Linguagem de programação     |
